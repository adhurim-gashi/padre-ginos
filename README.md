# Padre Ginos

A pizza ordering web app built while following along with Brian Holt’s React course.

## Features

- Browse available pizzas
- View pizza details
- Add pizzas to cart
- See items in the cart
- Submit the contact form
- Basic test setup with Vitest

## Tech Stack

- React
- Vite
- TanStack Router
- TanStack Query
- Vitest
- React Testing Library

## Purpose

This project was built for learning and practicing modern React concepts such as routing, server state, forms, and testing.
